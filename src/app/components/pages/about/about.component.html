Episode I

The beginning of development

This project has deveploment for trainning the concepts about development web, using Angular and Java with Spring Boot. The project was created based on the challenge proposed in github.com/AmeDigital/challenge-back-end-hit, where the idea was to create a CRUD capable of working with information from the https://swapi.co/.

I decided to create the front-end of the application to have the complete project. And that was the end result, I hope you like it and... May the Force be with you, always.
